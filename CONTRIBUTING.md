# Contributing

Thank you for considering contributing to this template! Please follow the guidelines below to help us maintain a high-quality project.

## How to Contribute

- Fork the repository and create your branch from `main`.

- Ensure code style and linting pass (`npm run lint` or `make format`).

- Add or update tests as appropriate.

- Document any new features or changes in the README.

- Open a pull request with a clear description of your changes.

## Code of Conduct

Please be respectful and follow the [Code of Conduct](../CODE_OF_CONDUCT.md).

A modern WXT + React + TypeScript template with testable domain logic, predictable structure, and best practices for CI, security, and customization.

## Quick Start

1. Copy `.env.example` to `.env` and update as needed.
2. Install dependencies: `make install`
3. Run the app: `make dev`
4. Run tests: `make test`
